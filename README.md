# NBA Game Performance Analysis (2002–2022)

## Overview

This project analyzes NBA game data from 2002 to 2022 to identify scoring trends and evaluate the impact of home-court advantage. Using Python and visual analysis, it explores how team performance changes across time, locations, and seasons.

## Dataset

Source: [Kaggle - NBA Games Dataset](https://www.kaggle.com/datasets/nathanlauga/nba-games/data)

Files used:
- `games.csv`
- `teams.csv`
- `game_details.csv` (optional)

## Tools and Libraries

- Python (pandas, numpy, matplotlib, seaborn)
- Jupyter Notebook
- Tableau (for dashboard visualization)
- PowerPoint (for presentation slides)

## Project Structure

- **notebook/** → Final Jupyter notebook  
- **figures/** → Exported visualization images  
- **presentation/** → PowerPoint slides  
- **summary/** → Final written report (docx)  
- **data/** → Input CSV files  

## Key Findings

- Home teams consistently outperform away teams.  
- Average total points per game have risen steadily since 2012.  
- Scoring remains consistent throughout the season with slight monthly variations.

## Recommendations

- Maximize home-court advantage through rest and preparation.  
- Focus on offensive efficiency and fast-paced play.  
- Manage fatigue and recovery for consistent performance.
